dolympia — Design Spec
📅 2026-04-04 ✅ Approuvé V1 MVP 📱 Mobile-first Next.js 14 · Supabase · Vercel
🎯 Contexte

dolympia est une application web de mini-jeux multijoueur en temps réel, mobile-first. Des joueurs rejoignent une session via un code (ou lien direct), s'affrontent sur une série d'épreuves, et accumulent des points.

Usage principal : jeu en présentiel — soirées, apéros — mais supporte également le jeu à distance. Hébergé sur Vercel.

🔄 Flow de partie
🏠 Accueil → 🚪 Lobby → 👥 Splash Équipes 5s si équipes → 🎮 Manches → 3s inter-manche → 🏆 Fin → 🔁 Rejouer

Lobby

Présence temps réel. Suggestion nb manches selon durée + joueurs. Mode équipe manuel : swipe gauche = rouge / droite = bleu.

Inter-manche

3 secondes avec classement partiel et points gagnés sur la manche.

✔️ Checklist de vérification
  • Lobby — 2 onglets → présence temps réel synchronisée
  • Lancement — host lance → 2 clients passent en /game/[code]
  • Manche — jouer quiz → scores affichés en inter-manche
  • Reconnexion — fermer onglet joueur → récupération état depuis DB
  • Équipes — mode auto → splash assignation + HUD équipes
  • Fin solo — podium top 3 + classement complet visible
  • Fin équipe — classement coloré par équipe (rouge / bleu)
  • Admin — accès sans auth → redirect /admin/login
  • Shake it — mobile iOS → permission DeviceMotion demandée
